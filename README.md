# Docker Compose Node.js web app

## setup
```
docker-compose up -d
```
http://localhost:3000/

## log
```
docker-compose logs -f

```
## MySQL
login
```
docker-compose exec mysql mysql -uroot -p
```
```
mysql> show databases;
mysql> use database;
mysql> show tables;
```
